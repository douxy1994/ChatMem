## 一、删除对话功能增强

### 1.1 确认对话框（前端）

**修改文件**：`src/App.tsx`

**做什么**：删除对话时，在用户选择完删除选项（OneDrive/WebDAV）之后、实际执行删除之前，弹出确认对话框。

**新增类型**：
```typescript
type DeleteConfirmState = {
  pending: true;  // 仅作为标志，显示确认对话框
} | null;
```

**新增状态**：
```typescript
const [deleteConfirm, setDeleteConfirm] = useState<DeleteConfirmState>(null);
```

**流程变更**：
- 旧流程：点删除 → 垃圾箱对话框 → 点"移到垃圾箱" → 直接执行删除
- 新流程：点删除 → 垃圾箱对话框 → 点"移到垃圾箱" → 确认对话框 → 点"确认删除" → 执行删除

**"移到垃圾箱"按钮变更**：
```tsx
// 旧代码
onClick={() => void confirmMoveConversationsToTrash()}

// 新代码
onClick={() => setDeleteConfirm({ pending: true })}
```

**新增确认对话框渲染函数**：`renderDeleteConfirmModal()`
- 显示要删除的对话列表
- 提示文字："此操作将删除本机记录和 OneDrive 同步记录，删除后无法找回"
- 两个按钮：取消 / 确认删除
- 确认后调用 `confirmMoveConversationsToTrash()`

**新增翻译字符串**：
```typescript
// 中文
```

### 1.2 前端 fallback 到 delete_memory_conversation

**修改文件**：`src/App.tsx` — `confirmMoveConversationsToTrash` 函数

**做什么**：当 `trash_conversation` 失败时（对话只在内存存储中），自动 fallback 到 `delete_memory_conversation`。

```typescript
for (const conversation of targets) {
  try {
    await invoke("trash_conversation", { ... });
  } catch (trashError) {
    // trash_conversation 失败，尝试 delete_memory_conversation
    console.warn("trash_conversation failed, trying delete_memory_conversation:", trashError);
    await invoke("delete_memory_conversation", {
      agent: conversation.source_agent || selectedAgent,
      id: conversation.id,
      deleteSyncBackup: shouldDeleteSync,
    });
  }
}
```

### 1.3 delete_memory_conversation 命令（后端）

**修改文件**：`src-tauri/src/main.rs`

**做什么**：新增 Tauri 命令，处理只存在于内存存储中的对话删除。

```rust
#[command]
async fn delete_memory_conversation(
    agent: String,
    id: String,
    _delete_sync_backup: Option<bool>,
) -> Result<(), String> {
    // 1. 总是从同步文件夹删除
    if let Ok(Some(settings)) = read_app_settings_from_disk() {
        let sync_folder = settings.sync.sync_folder.clone();
        if !sync_folder.is_empty() {
            let safe_name = local_sync::id_to_filename(&id);
            let sync_file = std::path::PathBuf::from(&sync_folder)
                .join("conversations").join(&agent).join(format!("{safe_name}.json"));
            if sync_file.exists() {
                let _ = fs::remove_file(&sync_file);
            }
        }
    }

    // 2. 从内存存储删除
    if let Ok(store) = MemoryStore::open_app() {
        store.delete_store_conversation(&agent, &id)
            .map_err(|e| format!("Failed to delete from memory store: {e}"))?;
    }

    Ok(())
}
```

**注册到 invoke_handler**：
```rust
.delete_handler(tauri::generate_handler![
    // ... 现有命令
    delete_memory_conversation,  // ← 新增
    // ...
])
```

### 1.4 trash_conversation 回退到同步文件夹

**修改文件**：`src-tauri/src/main.rs` — `trash_conversation` 函数

**做什么**：当适配器找不到对话时，尝试从同步文件夹读取。

```rust
let conversation = {
    let adapter = get_adapter(&agent)?;
    match adapter.read_conversation(&id) {
        Ok(conv) => conv,
        Err(_) => {
            // 适配器找不到 → 从同步文件夹读取
            let settings = read_app_settings_from_disk()?;
            let sync_folder = settings.as_ref().map(|s| s.sync.sync_folder.clone()).unwrap_or_default();
            if !sync_folder.is_empty() {
                let safe_name = local_sync::id_to_filename(&id);
                let file_path = std::path::PathBuf::from(&sync_folder)
                    .join("conversations").join(&agent).join(format!("{safe_name}.json"));
                if file_path.exists() {
                    let body = std::fs::read(&file_path).map_err(|e| format!("Read error: {e}"))?;
                    serde_json::from_slice::<Conversation>(&body).map_err(|e| format!("Parse error: {e}"))?
                } else {
                    return Err(format!("Conversation {id} not found in adapter or sync folder"));
                }
            } else {
                return Err(format!("Conversation {id} not found and no sync folder configured"));
            }
        }
    }
};
```

**同步文件夹删除逻辑**：
```rust
let should_delete_sync = delete_sync_backup.unwrap_or(false);
if should_delete_sync {
    if let Ok(Some(settings)) = read_app_settings_from_disk() {
        let sync_folder = settings.sync.sync_folder.clone();
        if !sync_folder.is_empty() {
            let safe_name = local_sync::id_to_filename(&id);
            let sync_file = std::path::PathBuf::from(&sync_folder)
                .join("conversations").join(&agent).join(format!("{safe_name}.json"));
            if sync_file.exists() {

                    Ok(()) => {

                        record.sync_backup_path = Some(sync_file.to_string_lossy().to_string());

                    }
                }
```

## 二、list_conversations 直接读取同步文件夹

**修改文件**：`src-tauri/src/main.rs` — `list_conversations` 函数

**做什么**：在从适配器和记忆库读取之后，额外从同步文件夹直接读取对话列表。

```rust
// 在记忆库读取之后添加
let settings = read_app_settings_from_disk().ok().flatten();
let sync_folder = settings.as_ref().map(|s| s.sync.sync_folder.clone()).unwrap_or_default();
if !sync_folder.is_empty() {
    let remote = local_sync::read_remote_conversations(std::path::Path::new(&sync_folder));
    for ((remote_agent, remote_id), (updated_at, body)) in &remote {
        if remote_agent != &agent { continue; }
        if seen_ids.contains(remote_id) { continue; }
        // 从 JSON 提取 project_dir, summary, msg_count
        let (project_dir, summary, msg_count) = if let Ok(val) = serde_json::from_slice::<serde_json::Value>(body) {
            let pd = val.get("project_dir").and_then(|v| v.as_str()).unwrap_or("").to_string();
            let sm = val.get("summary").and_then(|v| v.as_str()).map(|s| s.to_string());
            let mc = val.get("messages").and_then(|v| v.as_array()).map(|a| a.len()).unwrap_or(0);
            (pd, sm, mc)
        } else {
            (String::new(), None, 0)
        };
        seen_ids.insert(remote_id.clone());
        results.push(ConversationSummaryResponse {
            id: remote_id.clone(),
            source_agent: agent.clone(),
            project_dir,
            created_at: updated_at.clone(),
            updated_at: updated_at.clone(),
            summary,
            message_count: msg_count,
            file_count: 0,
        });
    }
}
```
